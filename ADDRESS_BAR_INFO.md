# 주소창 숨기기 가이드

## 핵심 답변

**브라우저에서 직접 접속해도:**

- ✅ **스크롤 시 주소창이 자동으로 숨겨집니다** (Android Chrome, Safari 등)
- ✅ 페이지 로드 시 약간의 스크롤로 주소창을 최소화합니다
- ⚠️ 하지만 완전히 없애려면 **홈 화면 추가가 필요합니다**

## 두 가지 방법

### 방법 1: 브라우저에서 그냥 사용 (주소창 부분적으로 보임)

- 모바일 브라우저에서 사이트 접속
- **스크롤하면 주소창이 자동으로 숨겨짐** (이미 구현됨)
- 화면 상단에 주소창이 약간 보일 수 있음

### 방법 2: 홈 화면에 추가 (주소창 완전히 숨김) ⭐ 추천

- Android: 메뉴 > "홈 화면에 추가" 또는 "앱 설치"
- iOS: 공유 버튼 > "홈 화면에 추가"
- 홈 화면 아이콘을 탭하면 **완전히 주소창 없는 앱처럼** 실행됨

## 기술적 이유

**완전히 주소창을 없애는 것은 보안상의 이유로 브라우저에서 직접 접속할 때는 불가능합니다.**

- 웹 사이트가 주소창을 완전히 제어하면 피싱 등의 보안 문제 발생 가능
- 따라서 PWA(홈 화면 추가) 방식만 완전한 주소창 숨김이 가능

## 현재 구현 상태

✅ PWA 설정 완료 (홈 화면 추가 가능)
✅ 스크롤 시 주소창 자동 숨김
✅ 모바일 뷰포트 최적화
✅ iOS/Android 모두 지원

## 사용자 안내

사용자가 원하는 경험에 따라:

- **간단히 브라우저에서 사용**: 스크롤 시 주소창이 자동으로 숨겨짐
- **완전한 앱 경험**: 홈 화면에 추가 안내
