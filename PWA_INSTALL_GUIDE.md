# 모바일에서 주소창 없이 사용하기 (PWA 설치 가이드)

## 중요 사항

**브라우저에서 접속할 때는 주소창이 보입니다.** 주소창을 완전히 없애려면 **"홈 화면에 추가"** 기능을 사용해야 합니다.

## 설치 방법

### Android (Chrome/Samsung Internet)

1. 브라우저로 `10x-test.vercel.app` 접속
2. 주소창 옆에 있는 **메뉴(⋮)** 클릭
3. **"홈 화면에 추가"** 또는 **"앱 설치"** 선택
4. 홈 화면에 추가된 아이콘을 탭하면 주소창 없이 실행됩니다!

### iOS (Safari)

1. Safari로 `10x-test.vercel.app` 접속
2. 화면 하단 **공유 버튼(⬆️)** 탭
3. **"홈 화면에 추가"** 선택
4. 아이콘 이름 확인 후 **"추가"** 클릭
5. 홈 화면 아이콘을 탭하면 주소창 없이 실행됩니다!

## 주소창을 동적으로 숨기기 (스크롤 시)

일부 브라우저에서는 스크롤 시 주소창이 자동으로 숨겨집니다.
이미 코드에 설정이 포함되어 있습니다.
